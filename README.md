## Overview

To use Lammps Python Wrapper in more Python-Like way

## Dependencies

* [Lammps Python wrapper](http://lammps.sandia.gov/doc/Section_python.html) ( *lammps* module )
* [*mpi4py*](http://pythonhosted.org/mpi4py/)

## Install

```
python3 setup.py install
```
