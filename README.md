## Overview

To use Lammps Python Wrapper in more Python-Like way.

## Dependencies

* [mpi4py](http://pythonhosted.org/mpi4py/)

## Install

```
python3 setup.py install
```

## Usage

```
python3 script.py
```

Please see "tests-and-examples" directory for example(s) of "script.py".
